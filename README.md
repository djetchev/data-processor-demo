# Data Processor

A simple Python utility for batch-processing customer records.

## Usage

```python
from data_processor import process_batch

records = [
    {"name": "Alice", "email": "alice@example.com"},
    {"name": "Bob", "email": "bob@example.com"},
]

result = process_batch(records)
```
